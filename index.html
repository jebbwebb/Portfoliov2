PROJECT

AUDIOPHILE

React/Redux/Router

CSS

Photoshop

ABOUT

This e-commerce store offers a complete shopping cart experience. To begin, simply click on the "Add To Cart" button!

The project was developed using React, Redux, and React Router. Redux was utilized to manage the shopping cart's state, while a custom hook was implemented to handle the state of the cart dropdown. React Router was employed for page routing, and CSS was used for styling throughout the entire project.

Initially, this project presented an immense challenge, pushing me to overcome one of the major hurdles I've encountered in my coding journey. Working with images in CSS had always been a struggle for me, and given the abundance of images on the website, I knew I had my work cut out for me. However, upon completing this project, I discovered a newfound fascination with the creative possibilities CSS offers, far beyond what most would typically associate it with. It has given me an appreciation for CSS as an art form, rather than merely a tool.
